
# PredictMyBlood

📌 Project Overview

PredictMyBlood is a machine learning–based system that predicts a person’s blood group using biometric signals.
The project uses Gabor signal processing for feature extraction and Convolutional Neural Networks (CNNs) for classification, providing an innovative and accurate method for blood group prediction.

🚀 Features

Biometric-based blood group prediction
Signal processing using Gabor filters
Classification using CNN
Novel approach compared to traditional methods

🛠️ Tech Stack

Programming Language: Python
Libraries/Frameworks: NumPy, Pandas, OpenCV, TensorFlow/Keras, Matplotlib
Environment: Jupyter Notebook / Google Colab

📊 Workflow

1. Data preprocessing and cleaning
2. Feature extraction using Gabor filters
3. Training CNN model
4. Evaluation and accuracy measurement
5. Predicting blood group of new samples
